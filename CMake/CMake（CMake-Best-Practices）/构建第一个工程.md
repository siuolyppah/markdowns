```sh
git clone https://github.com/PacktPublishing/CMake-BestPractices.git
cd CMake-Best-Practices/chapter_1
mkdir build
cd build
cmake ..
cmake --build .
```

>```



- Demo项目结构：

  ```
  .
  ├── CMakeLists.txt
  └── build
  └── src
  └── main.cpp
  ```

  

指定编译器：

1. 通过设置环境变量：

   ```sh
   CXX=g++-7 cmake /path/to/the/source
   ```

2. 设置CMake变量：

   ```sh
   cmake -D CMAKE_CXX_COMPILER=g++-7 /path/to/source
   ```
