# VS

要在Visual Studio 中构建CMake 项目，请先转到Project | Configure。这将调用CMake 配置步骤并生成所需的构建系统文件。配置完成后，单击Build | Build All 以生成项目。也可以通过F7 快捷键来触发Build All。注意，当保存CMakeLists.txt 文件时，Visual Studio 将自动重新配置，该文件是项目的一部分。
