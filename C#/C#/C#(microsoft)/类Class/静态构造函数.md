- 静态构造函数，每个类型仅执行一次。

- 要求：

  - 无参
  - 方法名与类型一致
  - static修饰

  ```cs
  class Foo
  {
      static Foo()
      {
          Console.WriteLine("Type Initialized");
      }
  }
  ```

  

- 在使用类型之前的一瞬间，编译器会自动调用类型的静态构造函数。
  - 实例化一个类型
  - 访问类型的静态成员
- 只允许使用 unsafe 和 extern 修饰符



- 初始化顺序：
  1. 静态字段的初始化器
  2. 静态构造函数
