# 强化阶段

数学：开始强化阶段的二轮学习，资料：武忠祥老师的高等数学辅导讲义、李永乐线性代数、基础阶段余炳森老师的书；习题集：880（这是我认为最好的习题集，没有之一）、书中例题、660、武老师每日一题。

408：继续基础阶段的一轮学习，在学习后面几本书的时候，可以回到前面几本书做一下剩下的选择题。结束基础阶段后，打印王道的思维导图，回顾所有的冲刺课知识点，在思维导图上写上笔记。

政治：7月中旬或8月初开始政治的学习，推荐徐涛老师的基础课程，但也不要全看，只有马原部分是必须看的。由于2022年会开展全国二十大，政治的中国特色社会主义部分知识点会有较大改动，可以先跳过这部分或者听考研政治老师的建议。听完每一章徐涛老师的课程，马上进行肖秀荣1000题，一开始会错很多，不要太担心。肖1000有刷题小程序，便于碎片时间的刷题，而且有错题整理和笔记，个人认为比纸质的好用，所以不购买肖1000也是可以的，肖1000我也不建议全部刷完。





# 真题

数学：建议把18年、20年、22年的三张试卷留到最后，从05年的真题开始，至少每两天完成一张真题试卷+批改+b站李艳芳真题讲解。资料：李永乐真题解析，普通的真题试卷、李林六套卷、李林四套卷（这十套模拟卷足够了）。

408：408的真题很少，做一套少一套，晚点开始刷是可以的，我大概在10月中旬才结束强化。我先把所有真题的大题做了一遍，因为能考大题的知识点是最重要最需要掌握的，留了近两年的作为最后的模拟。在刷完大题后，我才开始刷选择题。

政治：政治是唯一不用做真题的一门科目，这个阶段建议去听腿姐的技巧班，然后二刷肖1000题。资料：肖秀荣八套卷的选择、腿姐四套卷的选择。

冲刺阶段（直到考前最后一刻）

数学：二刷2010年后真题和几套优质的李林模拟卷，一天一套+批改；最后几天看b站考研竞赛凯哥的考前复盘。

408：二刷甚至是三刷，重视每一道题目；最后几天看b站计算机考研大师兄的考前陪伴。

政治：考前10天开始背肖四，建议跟空卡的带背视频，一天花三小时左右背诵。同时，整理选择题考点笔记，只有选择题做得好政治才能拿高分。
