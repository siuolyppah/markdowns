# 导入模块

```haskell
import Data.List
```

```sh
:m Data.List
```



仅导入模块中的特定函数：
```haskell
import Data.List(nub, sort)
```



隐藏模块中的特定函数：

```haskell
import Data.List hiding (nub)
```



限定导入：

```haskell
import qualified Data.Map
```

> 调用方式为：`Data.Map.filter`



限定导入并重命名：

```haskell
import qualified Data.Map as M
```

> 调用方式为：`M.filter`
